# Quantum Cryptography & Machine Learning Practicum
This repository documents my technical work in Quantum Information Theory, focusing on the intersection of secure communication (QKD) and Quantum Machine Learning (QSVM).
## 🚀 Key Projects
### 1. QKD Protocol Simulation
- Implemented **BB84 protocol** simulations using Qiskit.
- Analyzed the relationship between **QBER (Quantum Bit Error Rate)** and secret key length.
### 2. Quantum Support Vector Machine (QSVM)
- Utilized **SVM** algorithms for anomaly detection in quantum channels.
- Explored the use of **Quantum Kernels** to classify noise patterns, differentiating between environmental decoherence and potential eavesdropping.
